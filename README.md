## auto_payment_wht (Odoo 15)

โมดูลช่วยคำนวณภาษีหัก ณ ที่จ่ายจากใบ Invoice/Bill ที่เลือกบน Payment โดยรองรับการเลือกบางใบและแก้ไขฐานภาษี (ก่อน VAT) ต่อใบได้

### Dependencies
- account
- dtr_payment_invoice
- dtr_taxation
